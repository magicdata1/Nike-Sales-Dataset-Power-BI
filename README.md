# Nike US Sales Analysis

## Project Overview
An analysis of Nikeâ€™s sales performance across the USA. This project tracks profitability and sales efficiency using Power BI and Python.

## Key Analytical Insights

### 1. Regional Performance
* **The West** is the leading region with **$89.6M** in total profit.
* **The South** is the most efficient, boasting the highest average **Operating Margin (46.6%)**.

### 2. Sales Channel Efficiency
* **Online Sales** are the most profitable per unit with a **46.4% margin**.
* **In-store Sales** drive the highest total volume ($127.5M profit) but have the lowest margins (**35.6%**) due to physical overhead.

### 3. Product Performance
* **Nike Air Max 90** is the top "Hero Product," contributing **$7.3M** in operating profit.

## Data Structure
The dataset includes:
- **Location**: Region, State, and City.
- **Product**: Model names and IDs.
- **Financials**: Units Sold, Operating Profit, and Operating Margin.
- **Method**: In-store, Online, and Outlet.
